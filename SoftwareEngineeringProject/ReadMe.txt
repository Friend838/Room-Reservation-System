1. Source code 和 建置exe
我們是用unity 2019.4.9f1版本，用unity打開之後，在左下方有叫做Script的資料夾，點開來裡面就是我們的各個source code。
另外再scene的資料夾裡是我們的各個GUI。
建置exe的部分在左上方的File裡展開來，選擇Build and Run，設定好資料夾後，就會在裡面建置exe檔，名稱為SoftwareEngineering。

2. 直接開啟exe
在exe資料夾裡直接開啟名稱為SoftwareEngineering即可。
